## Введение

Default Uploader — как промежуточное программное обеспечение между CDN и вашим приложением. Эта функциональность позволит оптимизировать
ваши файлы самым простым способом.

## Настройка

<Steps>
  1. Создайте поддомен для ваших медиа файлов у вашего CDN провайдера и установить origin на api.defaultuploader.com.

  2. В вашем приложении установите адрес для ваших файлов с поддоменом **cdn**.
  3. Задайте настройки для оптимизации по умолчанию для всех файлов в личном кабинете.
</Steps>

Поддомен может быть как SECRET_KEY или другое значение, которое можно установить в личном кабинете.

## Пример

```
// было
<img src="https://site.com" />

// стало
<img src="https://cdn.site.com" />

// например, по умолчанию установлена опция [quality=75, format=auto]
Изображение было оптимизировано и преобразовано в оптимальный формат для бразуера с которого был сделан запрос
```
