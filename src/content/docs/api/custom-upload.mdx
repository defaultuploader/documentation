### ?path=PATH

To upload a file to a custom folder in your bucket, you need to add the **query** parameter **?path=PATH**.

export const exampleCodeCurl = `curl --location 'https://api.defaultuploader.com/v1/upload?path=mysite.com' \
--header 'Authorization: YOUR_SECRET_CLIENT_TOKEN' \
--form 'file=@"/C:/Users/UserName/Pictures/Saved Pictures/cat.jpg"'`;
export const fileNameBash = 'bash';

<Code code={exampleCodeCurl} lang="sh" title={fileNameBash} />

If you upload with the path **?path=mysite.com/cats**, your file structure in the bucket will look like this:

<FileTree>
- bucket/ — root folder of the bucket
  - mysite.com — custom path
    - cats — custom path
      - image — file type (image, video, audio, etc.)
        - jpg — file extension
          - file.jpg — file name

</FileTree>

<Aside title="Convenient Organization of Uploaded Files">
  You can organize files by unique user identifiers.
  This makes it easy to track how many resources each user consumes.

  <FileTree>
    - bucket/ — root folder of the bucket
      - alise@gmail.com // 5GB
        - cats/
      - bob@gmail.com // 125GB
        - cats/
        - dogs/

  </FileTree>
</Aside>
