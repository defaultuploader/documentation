### ?path=PATH

Для того чтобы загрузить файл в пользовательскую папку вашего бакета необходимо добавить **query** параметр **?path=PATH**

export const exampleCodeCurl = `curl --location 'https://api.defaultuploader.com/v1/upload?path=mysite.com' \
--header 'Authorization: YOUR_SECRET_CLIENT_TOKEN' \
--form 'file=@"/C:/Users/UserName/Pictures/Saved Pictures/cat.jpg"'`;
export const fileNameBash = 'bash';

<Code code={exampleCodeCurl} lang="sh" title={fileNameBash} />

Если загрузить по такому пути **?path=mysite.com/cats**, то ваша файловая структура в бакете
будет выглядеть следующим образом:

<FileTree>
- bucket/ — корневая папка бакета
  - mysite.com — кастомный путь
    - cats — кастомный путь
      - image — типа файла (изображение, видео, аудио, и др.)
        - jpg — расширение файла
          - file.jpg — название файла

</FileTree>

<Aside title="Удобная струтуризация загруженных файлов">
  Можно размещать файлы по уникальным идентификаторам пользователей.
  Так будет легко отслеживать, сколько расходуют ресурсов пользователи.

  <FileTree>
    - bucket/ — корневая папка бакета
      - alise@gmail.com // 5gb
        - cats/
      - bob@gmail.com // 125gb
        - cats/
        - dogs/

  </FileTree>
</Aside>
