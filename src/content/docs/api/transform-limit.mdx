В личном кабинете Default Uploader вы можете установить максимальное количество трансформаций для одного типа файла (изображение, видео).

Если количество трансформаций уже достигло ограничения, то при следующем запросе на трансформацию будет передан оригинал файла.

<Aside title="Надежная защита ваших ресурсов" type="tip">
  Строго рекомендуется установить значение.

  Это полностью защищает от злоупотреблением API.
  Злоумышленник не сможет заспамить ваш S3 такими запросами ?w=100, ?w=101 и тд.
</Aside>
