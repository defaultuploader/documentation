### Обнаружение нежелательного контента (NSFW)

> Нажмите кнопку **Сканировать** в личном кабинете

В результате сканирования будет сформирован текстовый файл с адресами файлов в S3, в которых был обнаружен NSFW контент.

Сканирование всего бакета занимает некоторое время, к вам на почту поступит уведомление о завершающем этапе сканирования,
это означает, что процесс завершится примерно через минуту.

Чтобы сканировать только часть бакета, введите префикс - это часть пути, с которого надо начать сканирование.

Чтобы просканировать папку dogs/ и все ее содержимое, включая вложенные папки,
сохраните следующий путь в поле ввода в личном кабинете:
<Code code="/mysite.com/dogs" lang="html" title="Введите префикс в бакете" />
<FileTree>
  - bucket/
    - mysite.com
      - **dogs/** — все файлы с этого уровня будут просканированы
        - **hot-dogs/**
          - ...
      - cats/
        - ...

</FileTree>

### Webhook на загрузку нежелательного контента (NSFW)
Всякий раз, когда вы будете загружать файл, ваш файл будет просканирован на NSFW контент.

Введите адрес вебхука, куда будет отправлено уведомление с подробной информацией об обнаруженном контенте.
<Code code="https://holy-dogs.com/top-secret-pics" lang="html" title="Введите адрес вебхука" />

Запрос будет отправлен методом POST без повторений. Так же эта информация нигде не сохраняется.
