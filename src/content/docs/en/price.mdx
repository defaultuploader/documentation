<Aside type="tip" title="Бонус при регистрации">
  Для нового аккаунта доступно 100 гб трафика и один час обработки файлов.

Default Uploader использует модель **"оплаты по факту использования"** (pay-as-you-go), где стоимость рассчитывается на основе потребленных ресурсов:

- **Трафик:** 1 гигабайт — 1 рубль. Трафик учитывается при загрузке или скачивании ресурса из хранилища.
- **CPU time:** 1 секунда обработки файла — 1 копейка.

<Aside title="Порог оплаты">
  На старте порог составляет 2000 рублей. Вы можете использовать сервис до достижения порога без оплаты и привязки карты.

  Этого хватит на 2ТБ трафика или 55 часов CPU time.
</Aside>

### Расчет стоимости для проекта с Default Uploader и S3 и CDN от Yandex Cloud

Для проекта с 200 ГБ трафика в месяц, 50 ГБ хранилища S3 и 3 600 секунд CPU времени расчет стоимости следующий:

- Допустим CDN обрабатывает 90% запросов: 180 ГБ × 0,96 руб/ГБ = 172,80 руб.
- Default Uploader (20 ГБ трафика + CPU): 20 ГБ × 1 руб/ГБ + 3 600 секунд × 0,01 руб/секунда = 56 руб.
- Yandex Cloud S3 (хранилище и операции PUT/POST/PATCH и др.): 98,49 руб. + 4,80 руб. + 1,95 руб. = 30,27 руб.

> Итого: 172,80 руб. (CDN) + 56 руб. (Default Uploader) + 105,24 руб. (S3) = **334,04 руб. в месяц.**
