Нажмите на **Проверить подключение** в блоке **Настройки для подключения S3 и CDN** в личном кабинете,
если вы увидели попап с успешным подключение, подздравляю!

Самое время загрузить файл и трансформировать его!

Есть несколько способов загрузить файл:
- Postman
- Curl
- Через интерфейс S3 хранилища
- и др.

В этом руководстве мы загрузим через Curl, другие способы можно будет найти на странице API.

> **YOUR_SECRET_CLIENT_TOKEN** найдите в личном кабинете

export const exampleCodeCurl = `curl --location 'https://api.defaultuploader.com/v1/upload' \
--header 'Authorization: YOUR_SECRET_CLIENT_TOKEN' \
--form 'file=@"/C:/path_to_file/file_example_JPG_100kB.jpg"'`;
export const fileNameBash = 'bash';
export const highlights = ['YOUR_SECRET_CLIENT_TOKEN', 'CMS'];

<Code code={exampleCodeCurl} lang="sh" title={fileNameBash} mark={highlights} />
