Вы можете выбрать любого провайдера S3 хранилища.

В большинстве случаев алгоритм одинаковый, нужно создать сервисный аккаунт, получить secret и access key и создать бакет.
**Endpoint url** и **region** можно найти в документации у провайдеров.

Для тестирования можно использовать локально развернутый MinIO с доступом в интернет.
