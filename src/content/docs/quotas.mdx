Service limitations. Quotas can be modified upon request to technical support.

If a file transformation request is made for a file that exceeds the quota, the original file will be returned.

### File Size
Below are the quotas for file size during transformation.

<Aside>
  Files of any size (up to 5TB) and format can be uploaded and downloaded, the quota only applies to files supported by transformation.
</Aside>

#### Images
The maximum allowed original image size for transformation is - **40mb**

#### Video
The maximum allowed original video size for transformation is - **4gb**
