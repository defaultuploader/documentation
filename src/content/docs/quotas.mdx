Сервисные ограничения. Квоты можно изменить по запросу в техническую поддержку.

Если отправить запрос на трансформацию файла, размер которого, больше чем в квоте — будет возвращен оригинал файла.

### Размер файла
Ниже приведены квоты для размера файла при трансформации.

<Aside>
  Загружать и получать файлы можно любого размера (до 5ТБ) и формата, квота действует только на поддерживаемые трансформацией файлы.
</Aside>

#### Изображения
Максимально допустимый размер оригинала изображения для трансформации - **40mb**

#### Видео
Максимально допустимый размер оригинала видео для трансформации - **4gb**
