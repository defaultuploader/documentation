Укажите в URL адрес файла и необходимые параметры для трансформации

```
Изображение будет скачено с https://hot-dogs.com/holy-pick.jpg и будет сделана оптимизация ?q_auto

https://api.defaultuploader.com/v1/image/fetch/66c78a3b3438de9506dd5e93/https://hot-dogs.com/holy-pick.jpg?q_auto
```
