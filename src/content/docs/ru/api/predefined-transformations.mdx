Создавайте правила трансформации при загрузках файлов, а также трансформации по умолчанию для всех файлов!

### Предопределенный набор файлов

```html
Это правило создаст набор изображений для разных размеров экрана и формата изображений.

['w=360', 'w=768', 'w=1024']
['w=360&f=avif', 'w=768&f=avif', 'w=1024&f=avif']
```

### Преобразование файла по-умолчанию

```html
Это правило включит преобразование файла по заданным настройкам без параметров.

['f=auto&q=75&w=768']
```
