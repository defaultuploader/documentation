Чтобы преобразовать изображение, необходимо сделать запрос к файлу с параметрами.
Сервис создаст новый файл, отправит его ответом на запрос и сохранит в ваше хранилище.

При следующих запросах с такими же параметрами, будет отдаваться уже созданный файл.

Пример:
```
// создать новое изображение с шириной 100px с сохранением пропорций
https://api.defaultuploader.com/v1/image/upload/token/sample.jpg?w=100
```
