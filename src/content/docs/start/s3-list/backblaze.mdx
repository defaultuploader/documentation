<Steps>
  1. <a href="https://www.backblaze.com/" target="blank">Go to Backblaze</a> and create an account.

  2. Go to the <a href="https://secure.backblaze.com/b2_buckets.htm" target="blank">bucket creation page</a>.
  3. Click on **Create a Bucket**.
  4. Enter the bucket name.
  5. Set Files in Bucket are: > **Public**.
  6. Leave the other settings as they are and click **Create a Bucket**.
  7. Go to the Application Keys tab and click **Add a New Application Key**.
  8. Save the values, **the secret key is shown only once**.
  9. Save the values, **the secret key is shown only once**.
  10. Save the **Access Key ID** and **Secret Key**, as well as the endpoint_url and region in your [dashboard](https://admin.defaultuploader.com).
</Steps>

<Aside title="File endpoint and Region">
  ```
  // it's best to check with the provider's documentation
  // the endpoint is indicated on your bucket's card
  endpoint_url: (example) https://s3.us-east-005.backblazeb2.com
  region: Check from the endpoint_url, "us-east-005" is the region
  ```
</Aside>
