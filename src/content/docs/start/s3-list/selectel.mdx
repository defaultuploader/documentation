#### Create a User

<Steps>
  1. Create an account on <a href="https://my.selectel.ru/profile/users_management/users?type=panel" target="blank">Selectel</a>.

  2. Create a <a href="https://my.selectel.ru/profile/users_management/users?type=service" target="blank">user</a>.
  3. Select **Service User**.
  4. Set the role to **Object Storage User**.
  5. Select your project from the list.
</Steps>

#### Create an S3 Key

<Steps>
  1. Go to the **Service Users** tab.

  2. Select the created user.
  3. Click **Add Key** in the S3 Keys section.
  4. Click generate and save the credentials.
</Steps>

#### Create a Bucket (Container)

<Steps>
  1. Go to the **Object Storage** -> **Containers** tab.

  2. Click the **Create Container** button.
  3. Make it **public**.
  4. In **Addressing**, check the box next to **Virtual-hosting**.
</Steps>

#### Configure Access Policy in the Container

<Steps>
  1. Go to the newly created container -> **Access Policy**.

  2. Click **Create Access Policy**.
  3. Choose a policy name, such as "default uploader policy".
  4. Set access to **allow**.
  5. Users: **Authorized** and select the created user.
  6. Action set: **All**.
  7. At the bottom, click **Save**.
</Steps>
