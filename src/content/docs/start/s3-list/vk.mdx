<Steps>
  1. <a href="https://mcs.mail.ru/" target="blank">Go to VK CLOUD</a> and create an account.

  2. Go to the Object Storage -> Buckets tab.
  3. Click on **Add**.
  4. Enter the bucket name.
  5. Default ACL setting: **public-read**.
  6. Leave the other settings as they are and click **Create a Bucket**.
  7. Go to the **Object Storage** -> **Buckets** tab.
  8. Click on the bucket name -> **Keys**.
  9. Click on **Add Key**.
  10. Enter a name and click **Create**.
  11. Save the **Access Key ID** and **Secret Key**, as well as the endpoint_url and region in your [dashboard](https://admin.defaultuploader.com).
</Steps>

<Aside title="File endpoint and Region">
  ```
  // it's best to check with the provider's documentation
  endpoint_url: https://hb.vkcs.cloud or https://hb.ru-msk.vkcs.cloud
  region: ru-msk
  ```
</Aside>
