You can choose any S3 storage provider.

In most cases, the process is the same: you need to create a service account, obtain the secret and access key, and create a bucket.
The **Endpoint URL** and **region** can be found in the provider's documentation.

<Aside type="tip">
  For testing, you can use a locally deployed MinIO with internet access.
</Aside>

Below are several guides for creating and connecting S3 storage, and the list will continue to grow.
